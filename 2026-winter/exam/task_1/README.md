
# 1 задание | 100 баллов

+ Ограничение времени: 1 секунда
+ Ограничение памяти: 256 МБ

## Вопрос

Дана строка s, которая содержит только цифры от 000 до 999, в которой не все элементы равны 000.
Нужно переставить в ней цифры таким образом, чтобы число, которое представляет данная строка, было
минимально возможным, а также чтобы в этой строке не было ведущих нулей.

Число содержит ведущие нули, если число не равно нулю, но при этом в начале числа стоит цифра 000.
Так, например, числа 01, 0303 содержат ведущие нули, тогда как 0, 30000 нет.

## Входные данные

В единственной строке задана строка s (1 <= ∣s∣ <= 9).
Гарантируется, что в этой строке есть элемент, отличный от 0.

## Выходные данные

В единственной строке выведите ровно одну строку — строку, которая
представляет собой минимально возможное число без ведущих нулей.

## Примеры данных

### Пример 1

+ Ввод: `682902`
+ Вывод: `202689`

### Пример 2

+ Ввод: `01`
+ Вывод: `10`
