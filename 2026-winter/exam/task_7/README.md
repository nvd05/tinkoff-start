
# 7 задание | 100 баллов

+ Ограничение времени: 1 секунда
+ Ограничение памяти: 256 МБ

## Вопрос

Девочка Аня пожелала на Новый год себе шахматную доску размера n * n.
И благодаря новогоднему чуду ее желание сбылось, да и дополнительно она получила k фигур шахматных слонов.

Аня не первый год занимается математикой, поэтому ее заинтересовал вопрос:
А сколько можно расставить на этой доске ровно k слонов так, чтобы не было двух слонов,
которые бьют друг друга по правилам шахмат. Но в информатике она слаба,
поэтому она просит вас посчитать это значение. Но так как ответ может
быть слишком большим, она просит его посчитать по модулю 10^9.

## Входные данные

В единственной строке заданы два числа `n, k (1 <= n <= 500, 1 <= k <= n^2)` - размер доски и количество слонов соответственно.

## Выходные данные

Выведите ровно одно число - количество расставить слонов на доску согласно условию по модулю `10^9 + 7`.

## Примеры данных

### Пример 1

Ввод:

```text
5 3
```

Вывод:

```text
1124
```

### Пример 2

Ввод:

```text
2 2
```

Вывод:

```text
4
```

### Пример 3

Ввод:

```text
4 3
```

Вывод:

```text
232
```
