
# 2 задание | 100 баллов

+ Ограничение времени: 1 секунда
+ Ограничение памяти: 256 МБ

## Вопрос

Дана строка `s`, состоящая только из латинских строчных букв.
Назовем строку хорошей, если в ней в качестве подстроки будут встречаться как строка `tbank`,
так и строка `study`.

Вам в качестве первой задачи на стажировку дали следующую задачу:
какое минимальное количество символов нужно поменять в строке, чтобы
получившаяся строка стала хорошей.

## Входные данные

В единственной строке задана строка `s (10 <= ∣s∣ <= 10^5)`.
Гарантировано, что строка содержит только строчные буквы латинского алфавита.

## Выходные данные

В единственной строке выведите ровно одно число - какое минимальное
число символов нужно заменить в старой строке так, чтобы встречались как
подстрока `tbank`, так и `study`.

## Примеры данных

### Пример 1

+ Ввод: `tbankstudy`
+ Вывод: `0`

### Пример 2

+ Ввод: `studtbankk`
+ Вывод: `5`
