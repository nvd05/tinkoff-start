
# 6 задание | 100 баллов

+ Ограничение времени: 2 секунды
+ Ограничение памяти: 64 МБ

## Вопрос

Вам дана строка `s`, состоящая из латинских строчных букв.
И вам нужно обрабатывать следующие операции над ней:

+ Даны два числа `l, r`.
  Вы берете подстроку `s_l s_{l+1} ..., s_r` и удваиваете ее и возвращаете получившуюся подстроку вместо изначальной в строке s.
  После этого размер строки s увеличится на длину этой подстроки.
  Так, если строка изначально была `abaca`, и вы выбрали l = 2, r = 4, то после данной операции строка s будет равна `abbaacca`.

+ Вам нужно сообщить, какой символ в строке на позиции i.

## Входные данные

В первой строке заданы два числа `n, q (1 <= n,q <= 10^5)` - исходный размер строки и количество запросов,
которые нужно обработать.

Во второй строке задана строка `s_1 s_2 ... s_n`​.
Гарантируется, что строка s содержит только латинские строчные буквы.

Далее в следующих q строках описаны запросы, которые необходимо обработать:

+ `1 l r (1 <= l <= r <= ∣s∣)` - вы должны удвоить подстроку `s_l s_{l+1} ... s_r`​.
+ `2 i (1 <= i <= ∣s∣)` - требуется вывести символ, который находится на позиции i в строке s.

Гарантируется, что `∣s∣ <= 10^{18}` после любого из запросов.

## Выходные данные

Для каждого запроса второго типа нужно вывести символ,
который находится в строке s на позиции i.

## Примеры данных

Ввод:

```text
3 9
abc
2 1
2 2
2 3
1 2 3
2 1
2 2
2 3
2 4
2 5
```

Вывод:

```text
a
b
c
a
b
b
c
c
```
