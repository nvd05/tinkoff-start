
# 4 задание | 100 баллов

+ Ограничение времени: 3 секунды
+ Ограничение памяти: 256 МБ

## Вопрос

Вам дан неориентированный граф.
Требуется сообщить длину наименьшего простого цикла в нём, либо же сообщить,
что в графе нет простого цикла.

## Входные данные

В первой строке содержатся два числа
n, m (2 <= n <= 2500, 1 <= m <= 5000) — количество вершин и рёбер в графе соответственно.

Затем в следующих m строках содержатся описания рёбер:

Каждое ребро задаётся двумя числами a, b (1 <= a,b <= n, a != b) - в графе есть ребро между вершинами a и b.

Гарантируется, что в графе нет петель и кратных рёбер.

## Выходные данные

В единственной строке выведите -1,
если в графе нет цикла,
иначе выведите длину минимального простого  цикла.

## Примеры данных

### Пример 1

Ввод:

```text
3 3
1 2
2 3
1 3
```

Вывод:

```text
3
```

### Пример 2

Ввод:

```text
6 7
1 6
6 4
4 2
2 5
5 6
2 3
3 5
```

Вывод:

```text
3
```
